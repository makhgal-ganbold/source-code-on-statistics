
# Python Virtual Environment үүсгэж идэвхжүүлэх

---

## Үүсгэх

Төслийн хавтас руу хандаж байгаад терминалаас дараах тушаал өгнө.

`python3 -m venv .venv`

## Идэвхжүүлэх

Үүссэн виртуал орчинг дараах тушаалаар идэвхжүүлнэ.

**Linux/MacOS**

`source .venv/bin/activate`

**Windows**

`.\venv\Scripts\activate`
